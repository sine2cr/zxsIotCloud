## 快速开始
1. 下载源码

```bash
git clone https://github.com/sine2cr/zxsIotCloud.git
```
2. maven 打包

```bash
mvn clean package -Dmaven.test.skip=true
```

3. 环境准备

JDK 1.8+

Maven 3.3+

Redis 6.2+

mysql 5.7+
